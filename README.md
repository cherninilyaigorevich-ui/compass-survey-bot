# Compass Survey Bot

Бот для мессенджера Compass, который автоматизирует опросы сотрудников.

## MVP

Первый сценарий:

- бот спрашивает сотрудника, менял ли он локацию;
- сотрудник отвечает "Да" или "Нет";
- при ответе "Да" бот просит указать новую локацию;
- ответы сохраняются в PostgreSQL;
- HR может смотреть статистику.

## Стек

- Python
- FastAPI
- PostgreSQL
- Docker
- Docker Compose
- GitHub Actions
